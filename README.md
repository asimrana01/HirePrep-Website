# HirePrep — AI Career Readiness Platform

Built for fresh graduates who want real interview feedback, skill gap analysis, and a learning roadmap — not generic advice.

---

## Setup in 4 Steps (local development)

### Step 1 — Firebase

HirePrep stores users and results in Cloud Firestore, and CV PDFs in
Firebase Storage. You need a Firebase project before the backend will run.

See **[DEPLOYMENT.md](./DEPLOYMENT.md) → Part 1** for the full walkthrough
(create project → enable Firestore → enable Storage → download a service
account key). It takes about five minutes.

### Step 2 — Backend

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
# Edit .env: add your Groq API key, the path to your Firebase service
# account JSON file, and your Firebase Storage bucket name
python app.py
```

Backend runs at: http://localhost:5000

### Step 3 — Frontend

```bash
cd frontend
python -m http.server 3000
# OR just open frontend/index.html directly in your browser
```

Frontend runs at: http://localhost:3000

### Step 4 — Use HirePrep

Open http://localhost:3000 in your browser. Sign up for a free account and start practicing.

## Features

- **AI Interview Practice** — Role-specific questions generated by Groq AI, with per-answer scoring and feedback
- **Adaptive Quizzes** — MCQ quizzes on any topic with instant explanations
- **CV Upload & Analysis** — Upload your PDF CV; AI extracts skills, education, and builds your profile; the original PDF is stored in Firebase Storage
- **Skill Gap Analyzer** — Paste a job description and see exactly what you're missing + a learning roadmap
- **Progress Tracker** — Every session saved to Firestore; track improvement over time

## Tech Stack

- **Frontend:** Pure HTML5 + CSS3 + Vanilla JS (no frameworks, no build step)
- **Backend:** Python 3.10+ / Flask, served by Gunicorn in production
- **AI:** Groq API (llama-3.3-70b-versatile + llama-3.1-8b-instant)
- **PDF Parsing:** pdfplumber
- **Database:** Cloud Firestore (users, results)
- **File Storage:** Firebase Storage (uploaded CV PDFs)
